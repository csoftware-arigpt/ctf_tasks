# КиберБудущее

Ремейк задачи **Код Судьбы** с отборочного этапа [BSUIR:CTF](https://t.me/bsuir_official/12768) на базе серверной части оригинальной задачи

## Что сделано?

- Переписан интерфейс
- Обновлен Flask
- Код приведен в более читаемый вид

## Как установить?

1. Создать Docker контейнер
2. Установить `python3`, `python3-venv`
3. Создать виртуальную среду
4. `pip3 install -r requirements.txt`
5. Создать флаг на сервере
6. Поменять параметры запуска
7. Запустить сервер
